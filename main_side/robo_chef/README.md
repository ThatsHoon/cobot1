## 팀원 브랜치 작업 순서

```bash
# 1. 저장소 클론 (최초 1회)
git clone https://github.com/dev-kibeom/robo_chef.git
cd robo_chef

# 2. 본인 이름으로 브랜치 생성
git checkout -b <본인이름>

# 3. 작업 후 커밋
git add .
git commit -m "커밋 메시지"

# 4. 원격에 푸시 (최초 푸시는 -u 필요)
git push -u origin <본인이름>
```
