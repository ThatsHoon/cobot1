ROBO CHEF / SECTION 07
S Strengths — 강점
  • 세그먼트 재생 모델 — 코드 변경 없이 레시피 확장
  • Firebase 실시간 동기화 — 고객/관리자 즉시 반영
  • 노드 2개로 단순화 — 유지보수성·안정성 향상
  • 3단계 무결성 가드 — 운영 안정성 확보
W Weaknesses — 약점
  • 녹화 품질이 조작자 숙련도에 의존
  • Firebase 무료 플랜 동시 연결 제한
  • 세그먼트 재녹화 시 재현성 편차 발생 가능
  • 단일 카메라 비전 피드백 미구현
O Opportunities — 기회
  • 메뉴 추가는 녹화+Firebase 등록만으로 가능
  • 비전 피드백 추가 시 적응형 조리 가능
  • 클라우드 확장으로 다중 로봇 관리
  • DualSense → 더 정교한 힘 제어 녹화
T Threats — 위협
  • Firebase 서비스 장애 시 전체 주문 시스템 중단
  • DSR_ROBOT2 API 버전 변경 시 재검증 필요
  • ROS 2 버전 업그레이드 시 호환성 리스크
  • 물리적 레이아웃 변경 시 전체 재녹화 필요
