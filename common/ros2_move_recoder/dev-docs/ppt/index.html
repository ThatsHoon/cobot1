02데모 ① · 관절(Joint) 좌표계

관절 단위로 정밀하게

관절 하나하나를 컨트롤러로 미세 조정해 자세를 잡고 녹화 → 그대로 재생됩니다.

무선 컨트롤러선 없이 로봇 곁에서 직접 조작
관절별 미세 조정6개 관절을 하나씩 골라 정밀하게
녹화 후 재생잡은 동작을 버튼 한 번으로 반복
JOINT MODE
관절 단위로 직접 미세 조정하며 녹화하고, 그대로 재생
03데모 ② · 작업(TCP) 좌표계

생각하는 방향 그대로

앞·옆·위아래(X·Y·Z) 직관적 이동으로 녹화 → 재생. 상황에 따라 관절 모드와 전환합니다.

작업 공간 기준로봇 끝(손)을 기준으로 이동
앞·옆·위아래사람이 생각하는 방향 그대로
두 모드 전환정밀 조정(관절) · 직관 이동(작업)
TCP MODE
앞·옆·위아래로 직관적으로 이동하며 녹화하고, 그대로 재생
04데모 ③ · 안전 제어

위험하면 멈추고, 확인 후 이어가기

재생 도중 버튼 한 번으로 멈추고, 안전을 확인한 뒤 다시 이어서 재생합니다.

즉시 일시정지재생 중 언제든 그 자리에서 정지
버튼 한 번 재개멈춘 지점부터 자연스럽게 이어서
비상정지·홈복귀위급 시 즉시 정지, 안전 자세 복귀
PAUSE / RESUME
재생 중 멈췄다가, 확인 후 다시 이어서 재생

② 자동 보정

손의 떨림과 잡음을 프로그램이 알아서 정리합니다.

흔들림부드럽게

③ 부드럽게 재생

한 줄한 번만 가르치면 — 보정은 자동, 재생은 무한 반복.
06핵심 가치

왜 이 도구인가

전문 지식 불필요

현장 작업자가 직접 가르치고, 직접 수정합니다.

빠른 티칭

움직여서 녹화하면 끝. 동작 변경도 즉시 반영됩니다.

집게까지 동기화

잡고 놓는 그리퍼 동작도 함께 녹화·재생됩니다.

안전 우선

즉시 정지·홈 복귀, 재생 중 일시정지/재개를 지원합니다.

사전 검증

실제 로봇 없이 가상 모드에서 먼저 확인하고 적용합니다.

조작 방식 자유

손으로 직접(매뉴얼 모드)이든, 무선 컨트롤러로든 — 편한 방식으로 가르칩니다.
